# Fahis
Fahis is a website built with the Django framework, it analyzes files
using Cuckoo sandbox.
# Features  
- Analyze files with Cuckoo Sandbox
- Account Management ( Register, Login, Logout, and
Reset password )
- Personal History
- Public Repository
- Public API.
