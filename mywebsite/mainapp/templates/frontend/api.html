فاحص يتيح للمستخدمين الاساتفادة من خدمة فحص الملفات عن طريق الAPI.
طريقة الاستخدام:
تنقسم العملية الى قسمين:
أولا, رفع الملف والحصول على المعرف. ويمكنك رفع الملف عن طريق الاتي:
import requests
REST_URL = "http://localhost:8000/api/upload"
file=open("file path","rb")
DATA={
    "environment":"environment you want to use",
}
r = requests.post(REST_URL,files={"my_file":file},data=DATA)

ثانيا, الحصول على نتيجة الفحص. ويمكنك الحصول على النتيجة عن طريق الاتي: 

import requests
REST_URL = "http://localhost:8000/api/result"
            
DATA={
    "resultID":task id
}
            
r = requests.post(REST_URL,data=DATA)
